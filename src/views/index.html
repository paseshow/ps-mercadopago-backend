Implements Api Mercado Pago in Paseshow

  • Versión: 0.1
  • Modification date:
  • Author: Equipo Desarrollo PaseShow

  • Cancelaciones

    • Sólo puedes cancelar pagos que estén en estado "pending" o "in_process".

  • Devoluciones

    • Sólo puedes hacer devolucion cuando el estado sea "approved".

    Puedes devolver un pago dentro de los 360 días desde su acreditación.

    Debes poseer suficiente dinero disponible en tu cuenta para devolver el monto del pago satisfactoriamente.

    Si tu comprador realizó el pago con tarjeta, la devolución será reintegrada en la misma. Si el pago fue realizado con otro medio, se reintegrará en la cuenta de Mercado Pago del comprador. En caso que no tenga una cuenta, crearemos una utilizando el e-mail que utilizó para realizar la compra.

  • Devoluciones parciales

    • Puedes realizar hasta 20 devoluciones parciales a un mismo pago.

Link referencia: https://www.mercadopago.com.ar/developers/es/guides/manage-account/account/cancellations-and-refunds
